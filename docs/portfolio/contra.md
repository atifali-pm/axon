## Profile bio (long — Contra lets you write a proper about)

I've built backend systems for 20+ years and spent the last year going deep on AI infrastructure. Axon, my public reference codebase, is the result: a production-ready AI agent SaaS that ships every layer a real product needs.

Where most AI SaaS tutorials stop at "chat with your docs", Axon is 9 integrated phases of real work:
- Multi-tenant Postgres with Row-Level Security enforced at the database (not just application filters)
- Queue-first async architecture (seven typed BullMQ queues)
- LangGraph agents with multi-LLM routing, tool calling, and streaming
- RAG pipeline with pgvector hybrid search and prompt-injection fences
- Agent marketplace (shareable agent configs)
- Android + iOS companion app (Expo SDK 52, voice, offline, push)
- Stripe billing with webhook idempotency
- Full observability (Prometheus + Grafana + Loki + Langfuse + ntfy alerts)
- Production deploy scaffolding for Oracle Cloud + Cloudflare Tunnel ($0/mo target)

Clients hire me to fork this into their stack, rebrand, and ship. Or to build a specific layer on their existing codebase. You get senior-architect work without the six-month rebuild.
